# مصر الذكية - الاستوديو الشامل (Smart Egypt)

منصة متكاملة للدردشة الذكية، توليد الصور، وتوليد الفيديوهات السينمائية باستخدام تقنيات الذكاء الاصطناعي، مع باك اند سحابي قوي مدعوم بـ **Appwrite**.

## 🚀 كيفية إعداد Appwrite Console

لضمان عمل المنصة بشكل صحيح، يجب إعداد مشروعك في [Appwrite Console](https://cloud.appwrite.io/) واتباع الخطوات التالية:

### 1. إنشاء المشروع والمنصة
- أنشئ مشروعاً جديداً باسم **Afnan**.
- أضف منصة **Web App**.
- في خانة **Hostname**، ضع رابط موقعك (مثلاً: `teslav1005.github.io`) لتجنب أخطاء CORS.

### 2. إعداد قاعدة البيانات (Database)
- أنشئ قاعدة بيانات باسم **SmartDB** واستخدم المعرف: `6a3da226000f35fb5466`.
- أنشئ **Collection** (جدول) باسم **chats**.
- أضف الحقول (Attributes) التالية في الجدول:
    - `userId` (String)
    - `title` (String)
    - `lastMessage` (String)
    - `createdAt` (DateTime)
    - `securityToken` (String)
    - `userToken` (String)
    - `subToken` (String)

### 3. إعداد التخزين (Storage)
- أنشئ **Bucket** جديداً واستخدم المعرف: `6a3d9658002685e5ea5e`.
- اضبط الصلاحيات (Permissions) للسماح للمستخدمين برفع الملفات.

### 4. إعداد المصادقة (Authentication)
- فعل تسجيل الدخول عبر **Google OAuth**.
- أضف روابط العودة (Redirect URLs) الخاصة بموقعك في إعدادات جوجل وAppwrite.

---

## 🌍 نظام اللغات (Localization)
المنصة تدعم اللغتين **العربية** و**الإنجليزية** بشكل كامل. يتم إدارة كافة النصوص عبر ملف `translations.json`. لإضافة لغة جديدة أو تعديل نصوص، قم بتعديل هذا الملف فقط دون المساس بكود الواجهة.

## 🔒 نظام الحماية
تعتمد المنصة على نظام توكنات مزدوج:
1. **Session Token**: لإبقاء المستخدم مسجلاً للدخول بأمان.
2. **Sub-Tokens (JWT)**: يتم توليد توكن فريد لكل طلب يخرج من المنصة ويتم إبطاله فوراً لمنع أي تلاعب عبر "الكونسول" في المتصفح.
